System.Numerics.Multiprecision Library

The System.Numerics.Multiprecision library is implemented by means of the GNU Multiprecision Arithmetic Library.

It contains three multiprecision classes: the BigInt class, the BigRational class and the BigFloat class.

There's overloads of the operators operator+, operator−, operator*, operator/, operator== and operator< for all multiprecision classes, and overloads of the operators operator%, operator&, operator^, operator| and operator~ for the BigInt class.
