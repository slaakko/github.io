System.IO Library

The System.IO library contains classes for doing input and output.

All external string input is expected to be in UTF-8 encoding and this library writes the string output as UTF-8 encoded.

Byte Streams

The ByteStream represents a stream of bytes that can be read from and written to. Some byte streams support also setting and getting the current stream position. The contents of a byte stream can be copied to another byte stream.

The FileByteStream is a byte stream that is connected to an underlying file. The MemoryByteStream is a byte stream that collects the data when written to, and the data is then returned when read from. The NetworkByteStream is a byte stream that is connected to an underlying TCPSocket. The BufferedByteStream is a buffering layer to another byte stream.

StreamWriter and StreamReader

The StreamWriter class is for writing data in text to an underlying ByteStream. The Write member function converts the data to be written to a UTF-8 encoded string and then writes it to the byte stream. The WriteLine member function first calls the Write member function to write the data and then writes a line ending.

The StreamReader class is for reading data from an underlying ByteStream. The Read member function reads a byte from the underlying byte stream and returns the byte read, or -1 if the end of the stream has been encountered. The ReadLine member function reads a UTF-8 encoded line from the underlying byte stream and returns it. You have to call the EndOfStream member function to tell if the end of stream has been encountered. The ReadToEnd member function reads the rest of data from the underlying byte stream and returns it as a UTF-8 encoded string.

BinaryWriter and BinaryReader

The BinaryWriter class is for writing data to an underlying ByteStream in big-endian binary representation. Strings are converted to UTF-8 encoding before writing and they end with a zero byte. If the underlying byte stream is a FileByteStream, it should be opened or created with the binary open mode.

The BinaryReader class is for reading big-endian encoded data from an underlying ByteStream. Strings are expected to be UTF-8 encoded and zero byte terminated. If the underlying byte stream is a FileByteStream, it should be opened with the binary open mode.

File

The File class is a static class that contains the following commonly needed file operations among others:

  • CreateText creates a UTF-8-encoded buffered text file without a byte order mark, opens it for writing and returns a StreamWriter containing it.
  • CreateBinary creates a buffered binary file, opens it for writing, and returns a BinaryWriter containing it.
  • OpenRead opens a buffered text file for reading and returns a StreamReader containing it.
  • OpenBinary opens a buffered binary file for reading and returns a BinaryReader containing it.
  • ReadAllText opens a buffered text file for reading, skips a possible UTF-8 byte order mark, reads the contents of it to a string and returns the string.
  • ReadAllLines opens a buffered text file for reading, skips a possible UTF-8 byte order mark, reads all lines it contains to a list of strings and returns the list.
  • Exists returns true if the given file name exists and false otherwise.

Path

The Path class is a static class for file path manipulation. It contains the following path operations among others:

  • Combine concatenates two paths.
  • ChangeExtension returns a path string with the given extension replaced.
  • GetDirectoryName returns the directory name portion of the given path.
  • GetFileName returns the file name portion of the given path.

GetFullPath

GetFullPath function returns a canonicalized absolute path for the given relative or absolute path.

GetCurrentWorkingDirectory

GetCurrentWorkingDirectory function returns the path to the current working directory as a canonicalized absolute path.
