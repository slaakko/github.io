Basic Facilities

Basic classes and functions are defined within the System namespace.

Strings

There are three string class specializations implemented using the same String class template:

  • string is for ASCII and UTF-8 strings,
  • wstring is for UTF-16 strings, and
  • ustring is for UTF-32 strings.

There's also conversion functions ToUtf8, ToUtf16, and ToUtf32 for converting between them.

An object of a basic type can be converted to a string with the ToString function, and be parsed from a string with functions starting with Parse.

Console

The Console class is for reading input from the standard input stream, and writing output to the standard output or standard error stream.

Date and Time

TimePoint and Duration

The TimePoint class represents a time since some epoch in nanoseconds. The epoch itself is unspecified. The Duration class is for computing a duration between two time points in nanoseconds, and for doing arithmetics with durations and time points. The Now function returns the current time point from the steady clock of the system.

Date

The Date class represents a date as a year, month and day. The Date.ToString member function converts a date to a string in format YYYY-MM-DD, or YYYYMMDD. The GetCurrentDate function returns the current date from the system clock. The ParseDate function parses a date from a string that must be in format YYYY-MM-DD or YYYYMMDD.

DateTime

The DateTime represents a date and a time as a Date and number of seconds since midnight. The DateTime.ToString member function converts a date and time to a string in format YYYY-MM-DDTHH:MM:SS where the dashes, colons, minutes or seconds can be omitted. The GetCurrentDateTime function return s the current date and time from the system clock. The ParseDateTime function parses a date and time string that must be in format YYYY-MM-DDTHH:MM:SS, where the dashes, colons, time part, minutes or seconds can be omitted.

Pseudo-random Numbers

The Random function returns next 32-bit pseudo-random number generated using Mersenne Twister pseudo-random number generator. Each thread has its own sequence of pseudo-random numbers. The pseudo-random number sequence is seeded from the system using the rand_s function so the seed should be cryptographically secure random number. The sequence will not be cryptographically secure itself, however, but see this FAQ. To obtain a 64-bit pseudo-random number, call the Random64 function. If a predetermined sequence of pseudo-random numbers are needed, call the InitRand with a seed before obtaining the first pseudo-random number.

Smart Pointers

UniquePtr

The UniquePtr class is a unique ownership smart pointer class. A unique pointer cannot be copied, but it can be moved.

SharedPtr

The SharedPtr class is a reference-counted smart pointer class.

Exception

The Exception class is the ultimate base class of all exceptions. Exception contains an error message and a stack trace. The ToString member function returns a string that consists of the full class name of the exception, an error message and a stack trace.
