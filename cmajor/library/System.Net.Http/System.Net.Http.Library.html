System.Net.Http Library

The System.Net.Http library contains very basic support for implementing a HTTP/1.1 client application. The fundamental classes are HttpClient, HttpHeaderCollections and UriReference.

Here's an example of connecting to site using HTTPS protocol and retrieving a HTML page from there. The program prints the following output:
